Welcome to py-qrsim-tcpclient's documentation!
==============================================

The `py-qrsim-tcpclient` is a Python implementation of the `QRSim TCP client`.
It allows to control the `QRSim Quadrotors Simulator
<https://github.com/UCL-CompLACS/qrsim>`_ with a Python interface.
